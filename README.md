## journal.js Functions

Below are the functions included in `journal.js` and what they do:

- `greet(name)`: Returns a greeting string containing the user's name.
- `cube(x)`: Returns the cube of a number (x * x * x).
- `isEven(n)`: Returns true if the number is even, otherwise false.
- `subtract(a, b)`: Returns a - b. If b is missing, returns the negative of a.
- `multiplier(factor)`: Returns a new function that multiplies numbers by a saved factor (closure example).
